# 🤝 دليل المساهمة

شكراً على اهتمامك بالمساهمة في مشروع دليل الحساسيات! يرجى اتباع الإرشادات التالية.

---

## 📋 المتطلبات

- معرفة بـ Flutter و Dart
- تثبيت Flutter SDK
- حساب GitHub
- معرفة بـ Git

---

## 🚀 خطوات المساهمة

### 1. Fork المشروع
```bash
# انقر على زر Fork على GitHub
```

### 2. استنساخ المشروع المنسوخ
```bash
git clone https://github.com/your-username/allergy-guide.git
cd allergy-guide
```

### 3. إنشاء فرع جديد
```bash
git checkout -b feature/your-feature-name
# أو للإصلاحات:
git checkout -b fix/your-bug-fix
```

### 4. إجراء التغييرات
- قم بالتعديلات المطلوبة
- اتبع معايير الكود (انظر أدناه)
- اختبر التغييرات

### 5. Commit التغييرات
```bash
git add .
git commit -m "وصف واضح للتغييرات"
# أمثلة جيدة:
# "إضافة ميزة إدارة الحساسيات"
# "إصلاح خطأ في ماسح الباركود"
# "تحسين الأداء في Firestore"
```

### 6. Push للفرع
```bash
git push origin feature/your-feature-name
```

### 7. فتح Pull Request
- انتقل إلى المشروع الأصلي
- اضغط "Compare & Pull Request"
- صف التغييرات بوضوح
- انتظر المراجعة

## 📐 معايير الكود

### أسماء المتغيرات
```dart
// ✅ جيد
String userEmail;
int productCount;
bool isLoading;

// ❌ سيء
String u;
int c;
bool loading;
```

### أسماء الدوال
```dart
// ✅ جيد
void initializeAuth();
Future<User> fetchUserData();
bool validateEmail(String email);

// ❌ سيء
void init();
void getData();
bool valid();
```

### التعليقات
```dart
// ✅ جيد
/// تحقق من صحة البريد الإلكتروني
/// 
/// [email] البريد المراد التحقق منه
/// يرجع true إذا كان صالحاً
bool validateEmail(String email) {
  return email.contains('@');
}

// ❌ سيء
void validate() { // التحقق
  // كود
}
```

### استخدام const
```dart
// ✅ جيد
const EdgeInsets.all(16.0);
const SizedBox(height: 20);

// ❌ سيء
EdgeInsets.all(16.0);
SizedBox(height: 20);
```

## 📝 تنسيق Commit

استخدم البادئات التالية:
- `feat:` لميزة جديدة
- `fix:` لإصلاح خطأ
- `refactor:` لتحسين الكود
- `style:` لتنسيق الكود
- `test:` لإضافة اختبارات
- `docs:` لتحديث التوثيق
- `chore:` للمهام الروتينية

**أمثلة:**
```
feat: إضافة التنبيهات للمنتجات الخطرة
fix: إصلاح تسرب الذاكرة في ماسح الباركود
docs: تحديث README بالإعدادات الجديدة
refactor: تحسين كود AuthService
```

## 🐛 الإبلاغ عن الأخطاء

### عند إيجاد خطأ:
1. تحقق أنه لم يتم الإبلاغ عنه مسبقاً
2. افتح Issue جديد
3. صف الخطأ بوضوح:
   - **الخطوات للتكرار**
   - **السلوك المتوقع**
   - **السلوك الفعلي**
   - **لقطات الشاشة** (إن أمكن)
   - **معلومات النظام** (النسخة، الجهاز، إلخ)

**قالب Issue:**
```markdown
## الوصف
[وصف واضح للخطأ]

## خطوات التكرار
1. [الخطوة الأولى]
2. [الخطوة الثانية]
3. [الخطأ يحدث هنا]

## السلوك المتوقع
[ماذا يجب أن يحدث]

## لقطات الشاشة
[إذا كان ذا صلة]

## معلومات البيئة
- نظام التشغيل: [iOS/Android/Web]
- إصدار Flutter: [الإصدار]
- الجهاز: [نوع الجهاز]
```

## 💡 اقتراح ميزة جديدة

### قبل البدء:
1. تحقق أن الميزة لم تُقترح مسبقاً
2. اسأل نفسك: هل هذه الميزة مفيدة للآخرين؟
3. افتح Discussion للنقاش

**قالب الاقتراح:**
```markdown
## الوصف
[وصف الميزة المقترحة]

## المشكلة المحلولة
[ما المشكلة التي تحلها هذه الميزة؟]

## الحل المقترح
[كيف تقترح تنفيذها؟]

## بدائل
[هل هناك بدائل أخرى؟]
```

## 🎨 واجهة المستخدم

عند إضافة شاشات جديدة:
- استخدم الألوان المعرفة في `config/app_colors.dart`
- تابع نفس نمط الثيم
- استخدم نفس حجم الخطوط والمسافات
- تأكد التوافقية مع الهواتف الصغيرة والكبيرة
